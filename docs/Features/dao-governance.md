# DAO & Governance

Skatehive operates as a decentralized autonomous organization (DAO). Community members participate in governance through voting and proposals.

---

## üèõÔ∏è Overview

Skatehive's governance is powered by two systems:
- **Nouns Builder** - On-chain governance on Base network
- **Hive Governance** - Witness voting and proposal discussions

Together, these create a skater-owned, skater-run community.

---

## üó≥Ô∏è How Governance Works

### Nouns Builder DAO

Skatehive uses [Nouns Builder](https://nouns.build/) for on-chain governance:

| Component | Description |
|-----------|-------------|
| **NFTs** | Each auction NFT = 1 vote |
| **Proposals** | On-chain proposals for treasury spending |
| **Voting** | NFT holders vote on proposals |
| **Treasury** | Community funds managed by DAO |
| **Execution** | Approved proposals execute automatically |

### Hive Governance

On the Hive blockchain:
- Vote for **witnesses** who run the network
- Participate in community **discussions**
- Stake **Hive Power** for influence

## üé® Skatehive NFTs

### Daily Auctions

Every day (or set interval), a new Skatehive NFT is minted and auctioned:
1. Unique, programmatically generated artwork
2. Auction runs for 24 hours
3. Highest bidder wins the NFT
4. All proceeds go to the treasury

### NFT Benefits

Holding a Skatehive NFT gives you:
- **Voting power** - 1 NFT = 1 vote on proposals
- **Governance rights** - Participate in DAO decisions
- **Community status** - Recognized as a stakeholder
- **Access** - Potential future benefits and airdrops

### Where to Bid

Visit [skatehive.app/auction](https://skatehive.app/auction) to:
- View the current auction
- See auction history
- Place bids with ETH

## üìù Proposals

### What Can Be Proposed?

Proposals can cover anything community-related:
- **Treasury spending** - Fund events, prizes, development
- **Community initiatives** - New features, partnerships
- **Parameter changes** - Auction timing, reserve prices
- **Social decisions** - Community guidelines, policies

### Creating a Proposal

1. **Discuss first** - Share your idea in Discord
2. **Gather support** - Get feedback from the community
3. **Draft the proposal** - Clear description and budget
4. **Submit on-chain** - Requires minimum NFTs or delegation
5. **Voting period** - Community votes over set timeframe
6. **Execution** - Approved proposals execute automatically

### Proposal Example

```
Title: Fund Local Skatepark Event

Description: 
Host a community skate event at LA Park with prizes.

Budget: 2 ETH from treasury

Breakdown:
- 1 ETH: Prize money for competition
- 0.5 ETH: Event supplies and permits
- 0.5 ETH: Video documentation

Timeline: 
Submit by Jan 1, Event on Feb 15
```

## üí∞ The Treasury

### What's in the Treasury?

The Skatehive treasury contains:
- ETH from NFT auction sales
- Donated funds from supporters
- Trading fees from Zora coins (2.5%)
- Other community contributions

### Treasury Addresses

| Network | Address | Purpose |
|---------|---------|---------|
| **Ethereum** | skatehive.eth | Hot wallet |
| **Base** | [Multisig](https://app.safe.global/transactions/queue?safe=base:0xc1afa4c0a70b622d7b71d42241bb4d52b6f3e218) | Main treasury |

### How Funds Are Used

Treasury funds are controlled by the DAO and used for:
- Community events and contests
- Platform development
- Marketing and growth
- Supporting skaters directly

## üó≥Ô∏è Voting

### On-Chain Voting

For Nouns Builder proposals:
1. Connect your wallet at [skatehive.app/auction](https://skatehive.app/auction)
2. View active proposals
3. Vote **For**, **Against**, or **Abstain**
4. Sign the transaction

### Voting Power

Your voting power depends on:
- Number of Skatehive NFTs you hold
- Delegated votes from others

### Delegation

Don't have an NFT but want to participate?
- Ask an NFT holder to delegate their vote to you
- Or, holders can delegate to active community members

## üîó Witness Voting (Hive)

### What is a Witness?

Witnesses are servers that run the Hive blockchain:
- Validate transactions
- Produce blocks
- Set blockchain parameters

### Why Vote for Skatehive Witness?

Voting for the Skatehive witness:
- Supports skateboarding on Hive
- Helps secure the network
- Shows community alignment
- Improves your leaderboard ranking!

### How to Vote

1. Visit [skatehive.app/witness](https://skatehive.app/witness)
2. Log in with Hive Keychain
3. Click "Vote" next to Skatehive
4. Confirm the transaction

## ‚ùì FAQ

### How much does an NFT cost?

Auction prices vary based on demand. Check the current auction at [skatehive.app/auction](https://skatehive.app/auction).

### Can proposals be rejected?

Yes. Proposals need majority approval and minimum participation to pass.

### What happens if a proposal fails?

It doesn't execute. The proposer can revise and resubmit based on feedback.

### Is the treasury safe?

The treasury uses a multisig wallet requiring multiple approvals for any transaction.

### Can I sell my NFT vote?

NFTs can be sold, and voting power transfers with ownership.
