## ğŸ—³ï¸ Nouns Builder

Skatehive uses **Nouns Builder** on Base network for NFT-based governance:

### How It Works

1. **Daily Auctions**: One NFT auctioned every 24 hours
2. **NFT = Vote**: Each NFT gives 1 vote in proposals
3. **Treasury**: Auction proceeds go to DAO treasury
4. **Proposals**: NFT holders can create and vote on proposals

### Participating

1. Win an NFT in a daily auction
2. Hold the NFT in your wallet
3. Vote on active proposals
4. Create proposals (if threshold met)

## â“ FAQ

### How do I get voting power?

Win NFTs in daily auctions on Nouns Builder.

### What's the witness vote for?

Witnesses run Hive nodes that process transactions. Voting supports network security.

### Where do auction proceeds go?

100% to the DAO treasury for community use.

### Can I submit a proposal?

NFT holders can create proposals if they meet the threshold.

### Is there a token?

Skatehive uses NFTs for governance, not a separate token. HIVE/HBD are used for rewards.
