### Step 1: Connect Wallet

Go to [skatehive.app](https://skatehive.app) and click **"Log In"** ‚Üí **"Wallet"**.

### Step 2: Choose Your Wallet

Select your wallet:
- **MetaMask** (most popular)
- **Rainbow**
- **Coinbase Wallet**
- **WalletConnect** (supports 300+ wallets)

<!-- TODO: Add screenshot of wallet selection -->

### Step 3: Approve Connection

Your wallet will pop up. Click **"Connect"** to link your address.

### Step 4: Sign the Message

Sign the verification message in your wallet. This proves you own the address.

**The message looks like:**
```
Skatehive wants to verify your wallet.
Address: 0x1234...
Nonce: abc123...
Issued at: 2026-02-23T...
```

**Important:** This signature does NOT give Skatehive access to your funds. It only verifies ownership.

## üîê Stay Logged In

### Browser Sessions

Your login lasts **30 days** by default. After 30 days, you'll need to log in again.

**To stay logged in longer:**
- Check "Remember me" when logging in
- Don't clear your browser cookies
- Keep Keychain/wallet extension installed

### Mobile App

On mobile, you stay logged in until you manually log out.

### Multiple Devices

You can be logged in on multiple devices simultaneously:
- ‚úÖ Desktop browser
- ‚úÖ Mobile browser
- ‚úÖ Mobile app
- ‚úÖ Tablet

Each device has its own 30-day session.

## üÜò Troubleshooting

### Email Magic Link Not Working

**Problem:** Link says "expired" or "invalid"
**Solution:** Links expire after 15 minutes. Request a new one.

**Problem:** Email not arriving
**Solution:** 
- Check spam/junk folder
- Wait 2-3 minutes (sometimes delayed)
- Try a different email address
- Contact support on Discord

### Hive Keychain Issues

**Problem:** Keychain not popping up
**Solution:**
- Make sure Keychain extension is installed and unlocked
- Refresh the page and try again
- Check if Keychain is disabled for skatehive.app
- Try using Hive Auth instead (mobile-friendly)

**Problem:** "Account not found"
**Solution:**
- Make sure you typed username correctly (no @ symbol)
- Check spelling carefully
- Try logging in on [hive.io](https://hive.io) first to verify account exists

### Wallet Connection Issues

**Problem:** MetaMask won't connect
**Solution:**
- Make sure MetaMask is installed and unlocked
- Try disconnecting other sites first
- Clear MetaMask cache (Settings ‚Üí Advanced ‚Üí Reset Account)
- Try using WalletConnect instead

**Problem:** "Wrong network" error
**Solution:**
- Skatehive works on Ethereum Mainnet
- Switch your wallet to Ethereum Mainnet
- Or use a different wallet that's already on mainnet

### Farcaster Login Issues

**Problem:** QR code won't scan
**Solution:**
- Make sure you're using the latest Warpcast app
- Try increasing screen brightness
- Take a screenshot and open in Warpcast
- Use the "paste link" option instead

## ‚ùì FAQ

### Do I need a Hive account to use Skatehive?
No! You can start with just an email. Upgrade to Hive later when ready.

### Can I link multiple wallets?
Yes! Link as many identities as you want (Hive, ETH, Farcaster, etc).

### What if I forget which method I used to sign up?
Try email login first. If that doesn't work, check your email for previous magic links.

### Is my wallet safe?
Yes. Skatehive never asks for your private keys or seed phrase. We only request signatures to verify ownership.

### Can I remove a linked wallet?
Yes, go to Settings ‚Üí Account ‚Üí Manage Connected Identities ‚Üí Disconnect.

### What happens if I lose access to my email?
If you've linked other identities (Hive, ETH), you can still log in with those. If not, contact support.
