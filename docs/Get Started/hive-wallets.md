# Which Hive Wallet Should I Use?

When you connect to Skatehive, you'll be asked to choose a **Hive wallet provider**. Each has different strengths depending on your needs.

---

## 🔑 Quick Comparison

| Wallet | Type | Best For | Security | Ease of Use |
|--------|------|----------|----------|-------------|
| **Keychain** | Browser extension | Desktop users | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **HiveAuth** | Mobile app (QR) | Mobile-first | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| **Ledger** | Hardware wallet | Maximum security | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| **PeakVault** | Browser extension | Alternative to Keychain | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |

---

## 🏆 Recommended: Hive Keychain

**Best choice for most users.**

### What Is It?
Browser extension (like MetaMask for Ethereum) that securely stores your Hive keys and signs transactions.

### Pros
- ✅ Most popular and widely supported
- ✅ Works on desktop browsers (Chrome, Firefox, Brave, Edge)
- ✅ One-click login and transactions
- ✅ Manages multiple accounts
- ✅ Shows transaction history
- ✅ Free and open source

### Cons
- ❌ Desktop only (no mobile version)
- ❌ Requires installing extension

### How to Set Up
1. Go to [Hive Keychain website](https://hive-keychain.com/)
2. Install the extension for your browser
3. Create or import your Hive account
4. Return to Skatehive and click **Keychain** when logging in

---

## 📱 HiveAuth - Mobile Solution

**Perfect if you want to keep keys on your phone.**

### What Is It?
Mobile app that stores your keys on your phone. You scan a QR code on your computer to authorize transactions.

### Pros
- ✅ Keys stay on your mobile device (more secure separation)
- ✅ Works with desktop sites via QR code
- ✅ Supports multiple accounts
- ✅ Available for iOS and Android

### Cons
- ❌ Requires phone nearby for every login
- ❌ Slightly slower than browser extension
- ❌ Less widely adopted than Keychain

### How to Set Up
1. Download **HiveAuth** app (iOS App Store or Google Play)
2. Import or create your Hive account in the app
3. On Skatehive, select **HiveAuth**
4. Scan the QR code with the app
5. Approve the login request

---

## 🔐 Ledger - Maximum Security

**For users with significant Hive holdings or paranoid security.**

### What Is It?
Hardware wallet (physical device) that stores your private keys offline.

### Pros
- ✅ **Highest security**: Keys never touch the internet
- ✅ Protects against computer malware
- ✅ Industry-standard cold storage solution
- ✅ Supports multiple cryptocurrencies

### Cons
- ❌ Costs $79-$150+ to purchase device
- ❌ More complex setup
- ❌ Slower for frequent transactions
- ❌ Overkill for casual users

### How to Set Up
1. Purchase a Ledger device (Nano S Plus or Nano X)
2. Set up the device following Ledger's instructions
3. Install Hive app on your Ledger via Ledger Live
4. Connect Ledger to your computer
5. On Skatehive, select **Ledger** and follow prompts

### Who Should Use It?
- Investors with large Hive holdings
- Security-conscious users
- Those familiar with hardware wallets

---

## 🔷 PeakVault - Alternative Extension

**Newer alternative to Keychain.**

### What Is It?
Browser extension developed by the PeakD team (popular Hive frontend).

### Pros
- ✅ Clean, modern interface
- ✅ Works similarly to Keychain
- ✅ Developed by trusted Hive team
- ✅ Supports multiple accounts

### Cons
- ❌ Less widely adopted than Keychain
- ❌ Fewer features than Keychain
- ❌ May not work with all Hive dApps

### How to Set Up
1. Install PeakVault extension from [PeakD website](https://peakd.com/)
2. Import or create your Hive account
3. Return to Skatehive and select **PeakVault**

## 🆕 Don't Have a Hive Account Yet?

You'll need a Hive account before using any wallet. Create one at:

- [Hive Onboard](https://hiveonboard.com/) - Free accounts with phone verification
- [Ecency](https://ecency.com/signup) - Free accounts via referral
- [PeakD](https://peakd.com/register) - Account creation with small fee

> 💡 **Tip**: Once you have an account, use Keychain if you're on desktop. It's the easiest path to start skating! 🛹

---

## 🔐 Security Best Practices

No matter which wallet you choose:

1. ✅ **Never share your keys** - Especially your **Master** or **Active** keys
2. ✅ **Use Posting Key for Skatehive** - The Posting key is enough for most actions
3. ✅ **Back up your keys** - Write them down on paper and store safely
4. ✅ **Use strong passwords** - For extensions and apps
5. ⚠️ **Beware of phishing** - Always verify you're on `skatehive.app`
