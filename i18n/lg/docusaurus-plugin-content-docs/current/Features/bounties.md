# Bounties

**Funa ssente nga okola tricks!** SkateHive Bounties ye enkola ya blockchain zombi omuli ba-skater bakola challenge za tricks ne bafuna emipeera mu crypto.

🎯 **Laba bounties:** [skatehive.app/bounties](https://skatehive.app/bounties)

---

## 🌐 Enkola Bbiri eza Blockchain

### Bounties za Hive
- **Ssente:** HBD oba HIVE
- **Platform:** Blockchain ya Hive (social layer)
- **Claims:** Replies mu posts n'evideo
- **Okukakasa:** Eyakitonda alonda abawanguzi manually

### Bounties za POIDH (Ethereum)
- **Ssente:** ETH
- **Platform:** Base oba Arbitrum (Ethereum L2s)
- **Claims:** On-chain via smart contract
- **Okukakasa:** Okulonda oba instant (kyesigamye ku config)

:::tip
Ebika byombi birabika wamu ku [/bounties](https://skatehive.app/bounties) — sengejja ne tabs **ALL / HIVE / POIDH**.
:::

## 🎯 Engeri Gy'ekola

### Ku ba-Claimers (Ba-Skaters)

1. **Tandika bounties** ku [skatehive.app/bounties](https://skatehive.app/bounties)
2. **Londa challenge** gy'oyagala okugezaako
3. **Kola trick** ogikubye
4. **Sindika clip yo:**
   - **Hive:** Nyiga "CLAIM IT" → Upload obujulizi → Sign n'ekikozesebwa Keychain
   - **POIDH:** Yungisa wallet → "SUBMIT CLAIM" → Upload → Kakasa transaction
5. **Funa** bwe kikakasibwa! 💰

### Ku ba-Creators (Ba-Sponsor)

1. **Nyiga "+ CREATE BOUNTY"**
2. **Londa blockchain:**
   - **HIVE:** Ku mipeera ya social/community
   - **ETH (POIDH):** Ku mipeera ya Ethereum L2
3. **Jjuza ebikwata:**
   - Ennyonnyola ya trick
   - Omuwendo gw'empeera
   - Olunaku lw'okuggwawo (optional, Hive yokka)
   - Media (ekifaananyi/video)
4. **Publish** olindirire submissions
5. **Kakasa abawanguzi** obalabe nga bafuna!

## 📝 Okutonda Bounty ya Hive

**Ebyeetaagisa:**
- Extension ya Hive Keychain etereezeddwa
- Account ne posting key

**Emitendera:**

1. Genda ku [skatehive.app/bounties](https://skatehive.app/bounties)
2. Nyiga **"+ CREATE BOUNTY"**
3. Londa **"HIVE"**
4. Jjuza foomu:
   - **Omutwe:** Ennyonnyola etereevu (okugeza: "Kickflip BS Boardslide")
   - **Ennyonnyola:** Amateeka, ebikwata ku spot (supports markdown)
   - **Empeera:** Omuwendo + Ssente (HBD oba HIVE)
   - **Deadline:** Olunaku (optional, kino kitandisa!)
   - **Ekifaananyi:** Upload thumbnail (optional)
5. Nyiga **"PUBLISH BOUNTY"**
6. Kakasa n'ekikozesebwa **Hive Keychain**
7. Kiwedde! Bounty erabika instant 🎉

**Engabanya y'Empeera (Hive):**
- **Omuwanguzi 1:** 100%
- **Abawanguzi 2:** 70% / 30%
- **Abawanguzi 3:** 50% / 30% / 20%

:::info Ekyokulabirako
Bounty ya 100 HIVE, abawanguzi 3 → 50 / 30 / 20 HIVE buli omu
:::

## 💎 Okutonda Bounty ya POIDH

**Ebyeetaagisa:**
- Wallet erimu ETH ku **Base** oba **Arbitrum** (Ethereum L2s)
- MetaMask, Rainbow oba wallet ekwatagana

**Emitendera:**

1. Genda ku [skatehive.app/bounties](https://skatehive.app/bounties)
2. Nyiga **"+ CREATE BOUNTY"**
3. Londa **"ETH (POIDH)"**
4. **Yungisa wallet** (singa teyungizibbwa)
5. Jjuza foomu:
   - **Erinnya lya bounty:** Omutwe omufupi
   - **Ennyonnyola:** Ebyeetaagisa ku trick
   - **Empeera:** Omuwendo mu ETH
   - **Chain:** Base oba Arbitrum (dropdown)
6. Nyiga **"CREATE BOUNTY"**
7. **Kakasa transaction** mu wallet yo
8. Lindirira ~30 seconds ku indexer
9. Bounty enaakolebwa! 🚀

:::tip Gas Fees
Kozesa **Base** ku misolo emikalu (~$0.01-0.05). Arbitrum nayo mikalu naye mtonnyeko.
:::

## 🏆 Okusaba Bounty

### Claim ya Hive

1. **Nyiga card** → Omuko gw'ebikwata gujja kuggulawo
2. **Nyiga "CLAIM IT"**
3. **Jjuza foomu:**
   - **Omutwe:** Erinnya lya claim yo
   - **Ennyonnyola:** Link ya video (YouTube, Streamable, IPFS)
   - **Upload obujulizi:** Video oba ekifaananyi
4. **Sign n'ekikozesebwa Hive Keychain**
5. **Lindirira okukebezebwa** — Ddayo ku muko gwa bounty

### Claim ya POIDH

1. **Yungisa wallet** (ku corner waggulu ku ddyo)
2. **Nyiga card** → Omuko gw'ebikwata gujja kuggulawo
3. **Nyiga "SUBMIT CLAIM"**
4. **Jjuza foomu:**
   - **Erinnya:** Ennyonnyola empi
   - **Ennyonnyola:** Link oba upload
   - **Tegeka media:** IPFS upload automatic
5. **Kakasa transaction** mu wallet
6. **Omutendera gw'okulonda** (singa enabled) oba instant approval
7. **Funa** bwe kikakasibwa! 💸

## ❓ FAQ

**Q: Nsobola okusaba bounty yange?**  
R: Nedda — bounties za ba-skaters abalala okuzikolera.

**Q: Bbanga ki okutuusa lwe nfuna ssente?**  
R: **Hive:** Eyakitonda bw'akakasa | **POIDH:** Okulonda bwe kuwanguka oba instant.

**Q: Singa claim yange esibiddwa?**  
R: Osobola okugezaako nate singa bounty ekyali open. Kebera amateeka otereeze submission yo.

**Q: Nsobola okusazaamu bounty?**  
R: **Hive:** Kyusa metadata ya post okuteeka nga closed | **POIDH:** Kozesa function ya cancel mu contract (ETH ekomawo minus gas).

**Q: Njawulo ki eri wakati wa Hive ne POIDH?**  
R: **Hive** = blockchain ya social ne community feel, emipeera HBD/HIVE, manual approval. **POIDH** = Ethereum smart contracts, emipeera ETH, on-chain voting/instant approval.
